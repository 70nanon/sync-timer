# SyncTimer

「待ち」を共有するシンプルなSNS。タイマーをポストとして投稿し、同じ瞬間を待つ人数をリアルタイムで共有できるWebアプリケーション。

## 📋 概要

SyncTimerは、タイマーを1つのポストとして扱う新しい形のSNSです。
「ラーメンができるまで3分」「配信開始まであと1時間」など、あらゆる「待ち」をポストし、
他のユーザーが「私も待つ」ボタンで同じ瞬間を待つことができます。
同時にゼロを迎える瞬間を共有する体験がコアコンセプトです。

### コアコンセプト
- **タイマー = 1ポスト**: 各タイマーは独立したポスト
- **同時待機人数の表示**: 同じ終了時刻を待つ人数をリアルタイム表示
- **「私も待つ」で参加**: ワンクリックで同じ待ちに参加
- **匿名参加**: ユーザー登録不要、ニックネームのみで参加可能
- **同時ゼロの瞬間**: カウントダウンが0になった瞬間の人数を表示

## 🚀 技術スタック

- **フロントエンド**: React 18 + TypeScript
- **ビルドツール**: Vite
- **データベース**: Firebase Firestore
- **バックエンド機能**: Firebase Cloud Functions（オプション）
- **日時処理**: date-fns
- **デプロイ**: Firebase Hosting
